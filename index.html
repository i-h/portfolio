Project TownBuilder

Android game / Java

Personal project

https://play.google.com/store/apps/details?id=org.rautasydan.townbuilder

The game is currently available only for testers. Pop me a message if you want in.

TownBuilder is a 2D, top-down building game which focuses on resource management, and later cooperation, to develop and level up a town that will keep you on the top of the leaderboards!

A pacifist co-op RTS, if you will.

The game is in its current form only a husk of what I hope it to be. The game was developed alongside the "Microtransactions in an Android Game" thesis, in which the design is described more in-depth.

/**
* Put a resource in the stash
* @param type The type to put in
* @param amount The amount to put in
*/
public void put(ResType type, int tier, int amount){
    Log.i(tag, "Putting " + amount + " pieces of tier "
        + tier + " " + type.name() + " to stash.");
    int i = type.ordinal();
    if(limits[i] > 0 || resources[tier][i] + amount <= limits[i]){
        resources[tier][i] += amount;
    } else {
        // Set resources to max and put rest to overflow
        int excess = (resources[tier][i]+amount) - limits[i];
        resources[tier][i] = limits[i];
        overflow[tier][i] += excess/overflowDivider;
    }
}
